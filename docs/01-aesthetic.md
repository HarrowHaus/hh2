# 01 · AESTHETIC (LOCKED)

## The one-liner
A real, heavily-customized Windows XP machine. It boots straight onto a desktop wearing a **dark scene-kid `.msstyles` skin**. It is unmistakably XP underneath (same Start menu, taskbar, title bars, window behaviors) — just dark-skinned, the way a power user tore Luna out in ~2004.

## Why XP-customization, not abstract design
In the XP era, *how you skinned your machine was your identity*: patch `uxtheme.dll` / StyleXP / WindowBlinds, custom `.msstyles`, BootSkin + LogonStudio, Samurize/DesktopX widgets, RocketDock, IconPackager, even shell replacement (LiteStep/bb4win), plus media-player customization (Winamp skins, foobar2000 Columns-UI layouts). The taste lives in **that customization layer**, expressed period-correctly — NOT by replacing XP's visual grammar. Recolored-Luna was lame; a unique dark rig was the flex.

## The palimpsest (structure, never narrated)
Three ownership layers, oldest→newest, felt through what's where:
- **A — family/factory:** untouched Luna Blue + Bliss, a `FAMILY` account, old cruft. (Surfaces when the user switches to the Luna visual style.)
- **B — teenage scene kid:** the dominant dark skin + the customization stack + the period software loadout. The personality.
- **C — 2026 owner:** current portfolio/work, sitting inside the rebuilt rig.
This *is* surface-order/depth-riot and the maturation arc. **It is never explained on screen** (see CLAUDE.md Rule 2).

## The customization stack (each ships as a real feature)
- **Visual styles** via Display Properties → `bug.msstyles` (dark, default) / **Luna Blue** / **Classic**. Tokens in `tokens/themes.css`. Switching persists.
- **Boot + logon** as themed surfaces (custom BootSkin-style boot; a LogonStudio-style logon with the accounts).
- **Dock** (RocketDock-style), **desktop system-monitor widget** (Samurize/DesktopX-style: CPU/RAM/now-playing/clock), **custom icon set**, **custom cursor**.
- **foobar2000** heavily customized (Columns UI dark layout) = the music pillar (see `03-content.md`).
- **Shell-replacement Easter egg** (deep, optional): a hidden alt-shell mode.

## Visual rules
- Match real XP layout/metrics exactly; only the colors/bitmaps change per visual style.
- Dark default: charcoal/black chrome, dark-red Start, dark window bodies, grim wallpaper (placeholder: band art / horror still).
- Luna: authentic XP blue title bars, green Start, Bliss-style wallpaper.
- Classic: 9x grey 3D + teal desktop.
- The dark skin's authoritative spec is `tokens/themes.css` (approved palette + the 3 styles). **Chrome geometry + behavior come from the winXP base — do not hand-build XP chrome or port it from any throwaway mock. Only the skin is ours.**
